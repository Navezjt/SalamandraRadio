Salamandra

desenvolvido por Matheus Carolino

O Salamandra é um software de automação gratuito de rádio para o Windows.

Projetado para rádios que precisam de um software simples, fácil e estável para programar suas transmissões.

Download (Instalador) Download (Portátil)

Compatível com Windows 7 SP1 + ESE (*), 8.1, 10 and 11.

(*) Seu Windows 7 deve estar completamente atualizado (com Service Pack 1 + Extended Security Updates) - caso tenha problemas com o Windows Update, procure como resolver o erro 80072EFE do Windows Update.

Funcionalidades

  • Player com suporte aos formatos de áudio mais comuns (wav, mp3, wma, ogg and flac)
  • Cartucheira com opções avançadas para reprodução de áudios avulsos.
  • Diversos modos para seleção das faixas, como aleatório, embaralhar ou manual para maior controle da programação.
  • Faixas especiais, como locução de horário, arquivos de áudio aleatórios de uma pasta, listas rotativas ou sequenciais para blocos de anúncios ou programas.
  • Um programador de eventos versátil, para agendar faixas, playlists, áudios, programas, em um dia e hora agendados ou de forma periódica, como dias da semana ou horas do dia.
  • Configurações de mixagem para grupos de arquivos, para aplicar diferentes parâmetros a músicas, comerciais, vinhetas, entre outros.
  • Um player simplificado para pré-escuta de faixas de áudio em outra saída de som.
  • Gerador de arquivos de log, para registrar qualquer atividade ou erros do player.
